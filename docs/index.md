# Documentation as a Deliverable

This “talk” is published as a set of simple web pages (GitHub Pages) so it can be:
- opened on **any** machine with a browser,
- copied and edited as **plain text**,
- and archived years into the future without a software license.

## 10-minute message

If someone had to continue your work tomorrow, they need two things:

1. **Evidence of what exists** (photos, measurements, versions)
2. **Instructions to reproduce it** (what, why, how, and how to verify)
